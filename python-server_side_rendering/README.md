Python - Server-Side Rendering

Description

This project explores server-side rendering (SSR) using Python. SSR generates
web pages (or, in this case, text files) on the server by combining a
template with dynamic data, rather than relying on client-side JavaScript to
build the content. The project starts with basic string templating and will
grow toward using Flask and Jinja to render HTML dynamically from JSON, CSV,
and SQLite data sources.

Learning Objectives


Understand server-side rendering and how it differs from client-side
rendering.
Learn the benefits of using SSR in web development.
Implement SSR in Python using the Flask framework.
Use the Jinja templating engine to dynamically generate HTML pages.
Read and display data from JSON, CSV, and SQLite sources.
Handle dynamic content and user input in web applications.
