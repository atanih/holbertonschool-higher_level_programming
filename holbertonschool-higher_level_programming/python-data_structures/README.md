# Python - Data Structures: Lists, Tuples

This project covers the fundamentals of data structures in Python, focusing on *lists, **tuples, **sequences, and built-in operations such as **list comprehensions, **tuple packing/unpacking, and the *del statement**.

## Learning Objectives

At the end of this project, you should be able to explain:

- What lists are and how to use them
- The differences and similarities between strings and lists
- The most common methods of lists and how to use them
- How to use lists as stacks and queues
- What list comprehensions are and how to use them
- What tuples are and how to use them
- When to use tuples versus lists
- What a sequence is
- Tuple packing and sequence unpacking
- The del statement

## Requirements

- Interpreted on *Ubuntu 20.04 LTS* using *python3 (version 3.8.5)*
- All files end with a new line
- First line of all files: #!/usr/bin/python3
- Code follows *pycodestyle* (version 2.7.*)
- All files must be executable
- A README.md at the root of the project folder is mandatory

| 10-divisible_by_2.py | Function that finds all multiples of 2 in a list. |
| 11-delete_at.py | Function that deletes the item at a specific position in a list. |
| 12-switch.py | Script that switches the values of a and b. |
