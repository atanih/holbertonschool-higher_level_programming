# Python - Classes and Objects

This project covers Object-Oriented Programming (OOP) in Python, focusing on classes, instances, attributes, and methods.

## Learning Objectives

- What is OOP and "first-class everything"
- The difference between a class, an object, and an instance
- Public, protected, and private attributes
- The _init_ method and how to use it
- Data Abstraction, Encapsulation, and Information Hiding
- Properties, getters, and setters the Pythonic way
- The _dict_ of a class and an instance
- How to use the getattr function

## Requirements

- Python 3.8.5 on Ubuntu 20.04 LTS
- pycodestyle 2.7
- All files must be executable
- Documentation is mandatory for all modules, classes, and methods
